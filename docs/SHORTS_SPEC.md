# Shorts 페이지 기획서

## 📌 개요

TikTok/Instagram Reels 스타일의 세로 스와이프 UI로 영어 단어를 학습하는 핵심 페이지.

---

## 🎯 목표

1. **몰입감 있는 학습**: 풀스크린 이미지로 단어의 의미를 시각적으로 각인
2. **중독성 있는 UX**: 틱톡처럼 자연스러운 스와이프로 계속 학습하게 유도
3. **다양한 시각화**: 10가지 아트 스타일로 같은 단어를 다르게 표현

---

## 🖼️ 화면 구성

### 현재 레이아웃
```
┌─────────────────────────┐
│  1/10          ↑↓ • ←→  │  <- 상단 (카운터, 힌트)
│                         │
│      Replenish          │  <- 단어
│    보충하다, 다시 채우다  │  <- 한국어 뜻
│  to fill or supply again │  <- 영어 뜻
│                         │
│   ┌─────────────────┐   │
│   │                 │   │
│   │     이미지       │   │  <- AI 생성 이미지
│   │   (정사각형)     │   │
│   │                 │   │
│   └─────────────────┘   │
│                         │
│         ● ○ ○ ○ ...     │  <- 하단 (이미지 인디케이터)
└─────────────────────────┘
```

### 스와이프 동작
- **↑↓ (세로)**: 다음/이전 단어로 이동
- **←→ (가로)**: 같은 단어의 다른 스타일 이미지로 이동

## 📊 데이터 구조

### API 응답 (GET /api/vocab/:word)
```json
{
  "word": "Replenish",
  "slug": "replenish",
  "meaning_en": "to fill or supply again",
  "meaning_kr": "보충하다, 다시 채우다",
  "images": [
    {
      "variation": 1,
      "style_id": "watercolor",
      "style_name": "수채화",
      "scenario": "A gardener pouring water...",
      "path": ".../replenish-01-watercolor.png"
    },
    ...
  ]
}
```

### 프론트엔드 상태
```typescript
interface ShortsState {
  currentWordIndex: number;      // 현재 단어 인덱스
  currentImageIndex: number;     // 현재 이미지 인덱스
  wordDataMap: Record<string, WordData>;  // 캐시된 단어 데이터
  bookmarks: string[];           // 북마크된 단어 slug
  progress: Record<string, 'known' | 'unknown'>;  // 학습 진도
}
```

## 🎨 디자인 가이드

### 색상
- 배경: #000000 (검정)
- 텍스트: #FFFFFF (흰색)
- 보조 텍스트: rgba(255,255,255,0.7)
- 인디케이터 활성: #FFFFFF
- 인디케이터 비활성: rgba(255,255,255,0.4)

### 타이포그래피
- 단어: h4, bold
- 한국어 뜻: body1
- 영어 뜻: body2, italic

### 이미지
- 비율: 1:1 (정사각형)
- 최대 너비: 500px
- 모서리: 16px radius
- 그림자: 0 8px 24px rgba(0,0,0,0.5)
